lab 1 Setup Laptop

Goals

  • To install multiple tools

Notes

  • Don’t get intimidated by the commands here. Just copy paste them! These are
    commands that you won’t have to remember again and just have to google.
  • When you see this

    Execute:

    It means you have to type it on ITerm2

Download ITerm2 (Mac only)

  • Download from https://www.iterm2.com/

Download Sublime Text 3 (Mac only)

  • Download from https://www.sublimetext.com/
  • Follow the instructions here https://olivierlacan.com/posts/launch-sublime-text-3-from-the-command-line/

Download homebrew (Mac only)

Execute:

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Download Anaconda

For Mac Users:

  • Download from: https://www.anaconda.com/download/#macos

For Linux Users:

  • Open ubuntu 18.04 terminal
  • install compilers / necessary libraries / software

Execute:

sudo apt-get install gcc gfortran git lsb
install Anaconda

Execute:

wget https://repo.anaconda.com/archive/Anaconda3-5.3.0-Linux-x86_64.sh
bash Anaconda3-5.3.0-Linux-x86_64.sh
