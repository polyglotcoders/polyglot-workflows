lab 2 Setup your bash profile

Goals

  • Create a bash profile / modify it if it exists
  • We do this section so we can have a prettified version of iterm2

What is a bash profile?

Where environment settings are stored. When a new iterm2 session starts, it will load the contents of the bash profile before executing commands.

Open up ~/.bash_profile

Execute:

sublime ~/.bash_profile

This should have opened up sublime text for you

Copy paste this into the text editor and save
